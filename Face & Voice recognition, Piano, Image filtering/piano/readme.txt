1. The organ program piano.m generates notes between C2-B3.
You can execute the program with “ >> piano “ from the main Command Window of
MATLAB.
To generate notes between C2-B2, press q,w,e,r,t,y,u from the keyboard (caps lock off)
To generate notes between C3-B3, press a,s,d,f,g,h,j from the keyboard (caps lock off)
To exit press ESC.
Also we used getkey.m file to get ascii values from the pressed buttons on keyboard.

2. We created jingle_bells.m program to generate well-known song Jingle Bells.
We created tone strings to generate the song, then we played this note string in the code.
You can execute the program with “>> jingle_bells”.
